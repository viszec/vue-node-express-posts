## setup backend API mongoDB
```
npm init
npm i express cors body-parser mongodb
npm i -D nodemon
npm install jshint
```
## setup frontend vue
```
npm i -g @vue/cli
vue create client
cd client
npm run serve
npm i axios
npm run serve
npm run build
```
